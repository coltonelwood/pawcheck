# PawCheck 🐾

**AI-powered pet health platform.** Photo → urgency assessment → vet referral. Plus training plans, nutrition plans, vaccine tracking, a vet finder, community features, and push notifications. Web-first PWA, paid subscription, ready to deploy.

Built with Next.js 14, Supabase, Stripe, Claude vision, and Google Places.

---

## Full deployment guide

### Step 1 — Supabase setup (~10 min)

1. https://app.supabase.com → New project (name it `pawcheck`)
2. Project Settings → API → copy:
   - Project URL → `NEXT_PUBLIC_SUPABASE_URL`
   - `anon` key → `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `service_role` secret → `SUPABASE_SERVICE_ROLE_KEY`
3. SQL Editor → New query → paste `supabase/migrations/001_initial_schema.sql` → Run
4. **Run migration 2:** SQL Editor → New query → paste `supabase/migrations/002_extended_features.sql` → Run
5. **Run migration 3 (for mobile push):** paste `supabase/migrations/003_expo_push_tokens.sql` → Run
6. **Run migration 4 (storage hardening):** paste `supabase/migrations/004_storage_hardening.sql` → Run. Locks photo buckets to 10MB max and image MIME types only.
7. **Run migration 5 (rate limits):** paste `supabase/migrations/005_rate_limit_rpcs.sql` → Run. Adds RPCs the API routes use to throttle AI calls per-user.
8. Authentication → URL Configuration:
   - Site URL: `https://pawcheck.app`
   - Redirect URLs: add `https://pawcheck.app/api/auth/callback` and `http://localhost:3000/api/auth/callback`

### Step 2 — Anthropic API (~2 min)

1. https://console.anthropic.com/settings/keys → create key → `ANTHROPIC_API_KEY`
2. Add $20+ credit so it doesn't cut off

### Step 3 — Stripe setup (~20 min)

1. https://dashboard.stripe.com (TEST mode while developing)
2. Products → Add product "PawCheck Premium"
   - Recurring price: $14.99 USD/month → copy price ID → `NEXT_PUBLIC_STRIPE_MONTHLY_PRICE_ID`
   - Add another recurring price: $79 USD/year → `NEXT_PUBLIC_STRIPE_YEARLY_PRICE_ID`
3. Developers → API keys:
   - Publishable → `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`
   - Secret → `STRIPE_SECRET_KEY`
4. Developers → Webhooks → Add endpoint:
   - URL: `https://pawcheck.app/api/stripe/webhook`
   - Events: `checkout.session.completed`, `customer.subscription.created`, `customer.subscription.updated`, `customer.subscription.deleted`, `invoice.payment_succeeded`, `invoice.payment_failed`
   - Copy signing secret → `STRIPE_WEBHOOK_SECRET`

### Step 4 — Resend (~5 min)

1. https://resend.com → API Keys → create → `RESEND_API_KEY`
2. Add and verify your sending domain (e.g. `pawcheck.app`)
3. `RESEND_FROM_EMAIL=PawCheck <hello@pawcheck.app>`

### Step 5 — Google Places API (~10 min) — for vet finder

1. https://console.cloud.google.com → Create project (or use existing)
2. APIs & Services → Library → enable:
   - **Places API (New)** — for vet search
   - **Geocoding API** — for zip code → coordinates
3. APIs & Services → Credentials → Create API Key → `GOOGLE_PLACES_API_KEY`
4. Restrict the key:
   - Application restrictions: HTTP referrers OR none (since we call server-side)
   - API restrictions: limit to Places API (New) + Geocoding API
5. **Cost note:** Places API costs ~$17 per 1,000 nearby-search calls. Set a daily quota limit in the GCP console. For a SaaS where most searches are one-per-session, budget ~$30-50/month at first.

### Step 6 — Web Push / VAPID keys (~3 min)

```bash
npx web-push generate-vapid-keys
```

Copy the output:
- Public Key → `NEXT_PUBLIC_VAPID_PUBLIC_KEY`
- Private Key → `VAPID_PRIVATE_KEY`
- `VAPID_CONTACT_EMAIL=mailto:hello@pawcheck.app`

### Step 7 — Cron secret

```bash
openssl rand -base64 32
```
→ `CRON_SECRET` (this protects `/api/cron/vaccine-reminders` from being hit by anyone)

### Step 8 — PostHog (~5 min, optional)

1. https://posthog.com → project API key → `NEXT_PUBLIC_POSTHOG_KEY`
2. `NEXT_PUBLIC_POSTHOG_HOST=https://us.i.posthog.com`

### Step 9 — Vercel deploy (~15 min)

1. Buy your domain (e.g. `pawcheck.app`)
2. Push code to GitHub
3. https://vercel.com → New project → import repo
4. Add all environment variables in Project Settings → Environment Variables
5. Deploy
6. Connect custom domain (Settings → Domains)
7. Update `NEXT_PUBLIC_APP_URL=https://pawcheck.app` and redeploy
8. **Vercel Cron is auto-configured** via `vercel.json` — runs `/api/cron/vaccine-reminders` daily at 14:00 UTC

### Step 10 — Smoke test

1. Sign up with real email → verify
2. Add a pet
3. Upload a test photo, confirm AI analysis renders
4. Add a vaccine with a due date in the past, then manually hit `/api/cron/vaccine-reminders` with the Bearer auth header — confirm reminder email arrives
5. Try the vet finder with your zip code
6. Make a community post, comment, like
7. Enable push notifications, send a test push
8. Trigger paywall (3 free queries) → upgrade in Stripe test mode → confirm subscription activates
9. Try a training plan and nutrition plan as Premium user

## The AI safety architecture

Three files define the legal moat:

- **`lib/prompts.ts`** — Photo-based urgency assessment. Mandatory vet-referral triggers (breathing, bleeding, poisoning, seizures, bloat, pale gums, trauma, urinary blockage in male cats, eye injuries, heatstroke, pregnancy complications). Confidence <60 → require vet visit. JSON output schema enforced.

- **`lib/prompts-training.ts`** — Positive-reinforcement only (no shock collars, no alpha rolls). Mandatory professional referral for aggression, biting, severe anxiety, resource guarding, compulsive behaviors, sudden onset.

- **`lib/prompts-nutrition.ts`** — AAFCO/WSAVA-based RER (70 × kg^0.75) × MER multipliers. Auto-recommends DACVN consultation for chronic disease, allergies, raw/homemade diets, pregnancy. Treats <10% rule. Toxic food list.

**Modify with attorney review.** Loosening any safety constraint increases your liability exposure.

## Common gotchas

- **Claude returns invalid JSON:** Already handled (strips markdown fences, validates fields). If you change prompts, retest.
- **Webhook not firing:** Check Stripe dashboard for delivery errors. Common causes: wrong URL, signature mismatch, route returning non-200.
- **Photo upload fails:** Check Supabase Storage policies. The migrations create public buckets `pet-photos` and `community-photos`.
- **`maxDuration` timeout:** AI routes have 60s timeout. Vercel Hobby caps at 10s — upgrade to Pro.
- **Push notifications on iOS:** Require iOS 16.4+ AND the user must install the PWA first ("Add to Home Screen"). Push on the iOS Safari tab itself does NOT work. The PWA install prompt addresses this.
- **Google Places quota:** Set a daily limit in the GCP console or one bad bot can run up a bill.
- **Cron job not running locally:** Vercel Cron only works in production. Test locally by manually hitting the URL with the Bearer header.
